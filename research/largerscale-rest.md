# Introduction

Given the many different operations required to be communicated among workers, a singular protocol is required to standardise communication.
Previously, communication was informal and dependent on whatever layer was used, but the specification of a standard will aid in taming the growth of possible operations to be communicated, once patterns become more obvious.
To this end, HTTP has been chosen, if for anything, it's ubiquity, with a complete [REST API available at swaggerhub](https://app.swaggerhub.com/apis/j-c/largerscale/2.0.0).
